A Halflife Deathmatch

Title                   : "High Rocky Mayhem"
Filename                : hrmayhem.bsp
Author                  : Beefster
Description             : High within the rocky cliffs, rests the test
			  of speed and survival.  A playground of death
			  and beef chunks awaits.  Think you're up to it.
 			  Sign up and say goodbye.  All exits are blocked
		   	  and the bloodshed has just begun.

Construction

Editor used             : Worldcraft v2.1
Known Bugs              : Didn't find any

================================================================

Installation

To play this level, extract the file "hrmayhem.BSP" into
your sierra\half-life\valve\maps directory and then start the game.
Start a deathmatch server and type "map hrmayhem" in the console.
